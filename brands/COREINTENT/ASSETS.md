## Usage Guidelines

- COREINTENT assets are used in developer-facing contexts: GitHub, API docs, technical READMEs
- Do not use in consumer-facing marketing for SongPal or MOSOKO
- Code Green should never be used on a light background — it is terminal-native
- All open-source releases from COREINTENT carry the MIT License
