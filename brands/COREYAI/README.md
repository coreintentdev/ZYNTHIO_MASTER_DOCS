## Overview

CoreeyAI is the AI engine that powers every intelligent feature across the Zynthio ecosystem. It is not a consumer product — it is the brain behind SongPal's composition tools, MOSOKO's adaptive learning, and COREINTENT's automation infrastructure.

CoreeyAI handles model selection, prompt engineering, API orchestration, and agentic workflows. It is designed to be modular: each brand pulls from the CoreeyAI layer for the intelligence it needs.

---

## Mission

To build a proprietary AI layer that gives every Zynthio product genuine intelligence — not off-the-shelf API calls, but purpose-trained, context-aware, creator-centric AI systems.

## Core Capabilities

- **Prompt engineering** — structured templates for consistent, high-quality outputs across all brands
- **Agentic workflows** — multi-step AI task automation (Perplexity Orb architecture)
- **Model orchestration** — route tasks to the optimal model based on context
- **Context management** — session restoration, state persistence across AI sessions
- **Creative AI** — music composition assistance, lyric generation, arrangement suggestions
