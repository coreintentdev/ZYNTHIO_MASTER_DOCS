## Overview

DJ Zynrose is the music artist persona of Corey McIvor — and the living proof of concept for the entire Zynthio ecosystem. Every track is created using the same tools Zynthio builds and teaches: SongPal, CoreeyAI, and the full creative-tech stack.

DJ Zynrose is not a side project. It is the cultural output of the ecosystem — the signal that proves the stack works, and the bridge between Zynthio's technology and a real creative audience.

---

## Mission

To create music that demonstrates what sovereign AI-assisted production sounds like — and to build an authentic artist community around the Zynthio ecosystem in the process.
